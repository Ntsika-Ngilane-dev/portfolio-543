# Calvin Portfolio

A modern, responsive personal portfolio website for Ntsika Ngilane, also known as Calvin. The site showcases his background, selected projects, contact information, a downloadable CV, and a floating AI-style chat assistant.

## Overview

This project is a polished single-page portfolio built with HTML, CSS, Bootstrap, and vanilla JavaScript. It is designed to be fast, elegant, and easy to customize for personal branding or freelance work.

## Features

- Responsive landing page and navigation
- About section with profile image, contact details, and personal summary
- Technology badges based on current development interests and GitHub profile
- Project showcase cards with live links
- Downloadable CV button
- Working contact form wired to email submission
- Floating, minimizable AI-style chat assistant
- Clean visual styling with a modern editorial aesthetic

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Font Awesome / Bootstrap Icons
- Playwright (for optional automation/testing tasks)

## Project Structure

- index.html — main structure of the portfolio page
- styles.css — styling and responsive layout
- chat.js — floating chat assistant behavior
- screenshots/ — project and profile images
- package.json — project metadata and dependencies

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Ntsika-Ngilane-dev/portfolio-543.git
   cd portfolio-543
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Open the project locally:
   - Open index.html directly in your browser, or
   - Use a live server such as VS Code Live Server

## Usage

You can customize the following areas easily:

- Personal bio and contact details in index.html
- Project cards and links in the Projects section
- Visual theme and spacing in styles.css
- Chat assistant responses in chat.js
